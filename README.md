# Memory Card

This Memory Card Game is a hands-on project built as part of The Odin Project's React Curriculum. The core objective is to practice declarative state management and side effects in React by building an interactive game application.

## Tech Stack

- Frontend Library: React (Vite-powered)
- Styling: CSS3 (Grid & Flexbox layout)
- API: PokeAPI (External REST data)
